# Img2Voxels
Deep Learning project that turns the 2D images into 3D voxels using TensorFlow framework.\
The project is applying the architecture of the VGG-16 encoder to take an input of RGB image and pass it to the 3D decoder.
## Model .ptl components best-model optimized and extracted using PyTorch-Mobile
